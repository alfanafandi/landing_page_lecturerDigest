Pratinjau Antarmuka Aplikasi

Eksplorasi pratinjau fitur utama aplikasi LectureDigest di bawah ini untuk melihat bagaimana aplikasi bekerja.

Merekam Kuliah Secara Real-Time

Sistem merekam audio kuliah dengan jernih, mendeteksi suara dosen, dan mempersiapkan data untuk diproses oleh kecerdasan buatan.

  • Perekaman audio berkualitas tinggi secara langsung.
  • Indikator status rekam yang informatif.
  • Penyimpanan otomatis langsung di database cloud Supabase.

Ringkasan Otomatis Berbasis AI

Menghasilkan rangkuman berstruktur dari transkrip kuliah panjang. Menemukan poin-poin terpenting dalam hitungan detik.

  • Pembuatan rangkuman otomatis berdasarkan materi kuliah.
  • Pengelompokan dalam sub-bab/topik yang rapi.
  • Ekstraksi poin-poin kunci materi secara detail.

Belajar Cepat dengan Flashcard & Kuis

Sistem mengekstrak konsep kunci dan membuat kuis serta pertanyaan kartu hafalan otomatis untuk melatih daya ingat Anda.

  • Metode Active Recall untuk mengingat materi kuliah.
  • Pembuatan kartu tanya-jawab otomatis dari teks ringkasan.
  • Kuis pilihan ganda untuk mengevaluasi pemahaman belajar.

Asisten Belajar Cerdas: DigestBot

Fitur tanya jawab langsung dengan AI pintar yang mengetahui isi rekaman kuliah Anda secara detail.

  • Tanya jawab interaktif seputar isi rekaman.
  • Minta penjelasan ulang pada bagian materi yang sulit.
  • Asisten AI pribadi yang selalu siap membantu kapan saja.

Unggah & Analisis PDF

Membaca dan merangkum slide kuliah/PDF bahan ajar Anda langsung di dalam aplikasi.

  • Unggah file bahan kuliah berupa slide presentasi atau dokumen PDF.
  • AI akan mengekstrak informasi dan membuat materi ringkasan.
  • Bahan ajar terpusat untuk memudahkan belajar terarah.

Diagnosis Pemahaman & Remedial Bot

Mendiagnosis kelemahan belajar Anda dan memberikan bantuan remedial otomatis.

  • Analisis otomatis tingkat pemahaman per topik kuliah.
  • Remedial Bot memberikan materi remedial terarah.
  • Rekomendasi materi yang perlu dipelajari kembali agar nilai maksimal.

Cara Instalasi Aplikasi

Pilih platform perangkat Anda untuk melihat panduan instalasi aplikasi LectureDigest.

1

Unduh File APK

Klik tombol unduh untuk mengunduh file app-release.apk (~60 MB).

2

Izinkan Instalasi

Buka file APK. Jika muncul peringatan, masuk ke Pengaturan dan aktifkan Izinkan instalasi dari sumber tidak dikenal.

3

Pasang & Jalankan

Ketuk Pasang (Install) dan tunggu proses selesai. Buka aplikasi dan mulai belajar!

1

Unduh Folder Rilis ZIP

Klik tombol unduh untuk mendapatkan file kompresi ZIP (misalnya Release.zip) dari Google Drive.

2

Ekstrak File ZIP

Setelah selesai diunduh, klik kanan pada file ZIP tersebut dan pilih Extract All... (Ekstrak Semua) ke folder pilihan Anda.

3

Jalankan Aplikasi

Buka folder hasil ekstrak, cari file lecturer_digest.exe, lalu klik dua kali untuk menjalankan aplikasi!
